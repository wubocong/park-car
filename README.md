# Cocos2dx 停车游戏

使用 Cocos Creator 1.9.3 开发

[实例演示](https://park.wubocong.com/)

## 项目结构

```
assets
  |--i18n（国际化）
  |--migration（迁移过程）
  |--res（使用的资源）
  |   |--font（字体）
  |   |--textures（贴图资源）
  |--scripts（公共脚本）
settings （编译设置）
```

共有三个场景（`start.fire`,`first.fire`,`second.fire`），分别为开始，第一关（侧方停车）及第二关（倒车入库）
